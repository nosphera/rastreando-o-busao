Para rodar:

Pre-requisitos : 
node/npm /react-native-cli , e o android SDK 26 se for rodar no android

baixe os fontes;

rode o comando "npm install";

inicie o emulador ou conecte o celular;

execute "react-native run-android" ou "react-native run-ios"
